发明名称--一种云计算中实现隐私保护的轻量级同态加密方法

摘要:

本发明涉及安全云存储和计算，特别涉及一种云计算中实现隐私保护的轻量级同态加密方法，包括：生成密钥；进行加密；利用自持私有的私钥解密过程；其中，加密和解密过程可以分别用两大通式来简单地总结：加密过程，c＝g^m*r^n mod n^2；解密过程，m＝L(c^lambda mod n^2)*u mod n；u＝(L(g^lambda mod n^2))^(‑1)mod n。本发明的定位是对数据进行轻量级加密，保证其同态性，使得数据在不解密的情况下实现加法运算，而能保证结果不变。
